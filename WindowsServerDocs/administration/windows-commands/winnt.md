# <a name="winnt"></a>winnt



O Winnt foi preterido e não há garantia de que haja suporte em versões futuras do Windows.

Essa ferramenta está incluída no Windows Server 2003. Para obter mais informações, consulte [WinNT](https://technet.microsoft.com/library/cc755763(v=ws.10).aspx).
